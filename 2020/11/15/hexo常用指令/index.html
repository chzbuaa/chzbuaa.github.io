hexo常用指令

发表于 2020-11-15

改动完之后上传：

hexo clean 清理上次静态文件
hexo s -p 5555(去localhost:5555看一眼)
hexo g  产生静态文件
hexo d  上传github

写完新文章或者改动必须清理本地上次生成的静态文件，之后才可以上传GitHub

hexo clean

新建网站chzblog:

hexo init chzblog

写新的文章”如何建博客”：

$ hexo new "如何建博客"

查看本地生成博客：http://localhost:5555/

$ hexo s -p 5555

生成静态文件：

$ hexo g

上传至github:

$ hexo d

新建类别：

$ hexo new page categories

新建标签：

$ hexo new page tags
